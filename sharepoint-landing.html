Step-by-Step Guides
Click-by-click instructions for common tasks
Expand any guide below and follow along directly in SharePoint.
1
How to Edit & Publish a Page
▶
  1. Navigate to the SharePoint page you want to edit.
  2. In the top-right area, click Edit (pencil icon). The page will enter edit mode with a toolbar at the top.
  3. Click on any text section to start editing. A formatting toolbar will appear above.
  4. To add a new section or web part, hover between sections and click the + icon that appears.
  5. To add an image, click + → choose Image → upload from your device or OneDrive.
  6. When done, click Republish (top-right) to save and publish your changes live.
  7. If you only want to save as draft, click Save as draft instead.
2
How to Upload Files to a Document Library
▶
  1. Go to your SharePoint site and click Documents in the left navigation (or Site Contents → Documents).
  2. Click Upload in the top menu → choose Files or Folder.
  3. Browse and select the file(s) from your computer. They will upload and appear in the library.
  4. To organise, click + New → Folder to create folders first, then drag files into them.
  5. To rename a file, click the three dots (…) next to the file → select Rename.
3
How to Share a File or Folder
▶
  1. In your Document Library, hover over the file and click the three dots (…) next to it.
  2. Click Share. A sharing panel will appear on the right.
  3. Under "Send Link", choose who can access: People in your organisation (recommended) or Specific people.
  4. Type the name or email of the person you want to share with.
  5. Set permission level: Can view (read-only) or Can edit.
  6. Click Send — they'll receive an email with the link.
4
How to Post a News Article
▶
  1. Go to your SharePoint site homepage.
  2. Click + New in the top command bar → select News post.
  3. Choose a layout template (e.g. Blank, Visual, Side-by-side).
  4. Click the title area to type your news headline.
  5. Click below the title to add body text, images, or links using the + web part button.
  6. Add a thumbnail image by clicking the camera icon in the top banner area.
  7. When ready, click Post news (top right) — your news item will appear on the homepage news feed.
5
How to Create a List
▶
  1. On your site, click + New → List.
  2. Choose Blank list or start from a template (e.g. Task tracker, Issue tracker).
  3. Give the list a name and click Create.
  4. To add columns, click + Add column on the right of the header row → choose type (Text, Date, Choice, etc.).
  5. To add a new row/item, click + New item → fill in the fields → click Save.
  6. To filter or sort, click the column header → use the dropdown options.
6
How to Update the Site Navigation
▶
  1. You must be a Site Owner or have edit permissions. Go to your site.
  2. Click Edit at the bottom of the left navigation panel (you'll see a pencil icon).
  3. To add a link, click + Add link → paste the URL and set a display name → click OK.
  4. To reorder items, drag the ⠿ handle to the left of each item up or down.
  5. To remove a link, hover over it → click the three dots (…) → select Remove.
  6. Click Save when done. Changes are live immediately.
Pro Tips
Quick wins for everyday use
$ tip --shortcut
Press Ctrl + E on any SharePoint page to instantly jump into Edit mode without hunting for the button.
Keyboard shortcut
$ tip --search
Use the Search bar at the top of any site — it searches files, pages, and people across your entire SharePoint environment.
Search
$ tip --version
SharePoint keeps version history for all files. Right-click a file → Version history to restore a previous version.
File recovery
$ tip --coauthor
Multiple people can edit the same Word/Excel file at once. Open the file → click Edit in Browser to co-author in real time.
Collaboration
$ tip --pinned
Pin frequently used sites: go to the SharePoint home → hover a site card → click the pin icon to keep it at the top.
Organisation
$ tip --alert
Get email alerts when files change: select a file → … → Alert me → set frequency (immediately / daily / weekly).
